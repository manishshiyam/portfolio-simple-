# Personal Portfolio Website

## 📌 Overview
This is a simple personal portfolio website created using HTML and CSS. The website consists of two pages and is designed to present my profile in a clean and responsive layout.

- Responsive design
- Clean and modern layout
- Two-page website
- Easy navigation
- Built using only HTML and CSS

## 🛠️ Technologies Used
- HTML5
- CSS3

```

```

## 🎯 Purpose
The purpose of this project is to practice front-end web development and create a personal portfolio using HTML and CSS.
